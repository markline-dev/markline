## Install

```bash
npm install markline
npx markline init
```

## Develop

```bash
npx markline dev
```

Open the printed URL and edit files under `docs/`. Update `docs.json` to change
navigation, branding, and the API base URL.

## Ship

```bash
# Static HTML for any CDN / GitHub Pages / S3:
npx markline export

# Or a Node server for Docker / Vercel:
npx markline build && npx markline start
```
