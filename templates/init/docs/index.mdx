## What is this

This is a starter docs site generated by `markline init`. Edit the Markdown
files in `docs/`, describe your navigation and branding in `docs.json`, and run
`markline dev` to preview.

Markline renders MDX, highlights code, and turns an OpenAPI spec into a full,
interactive API reference — with a live "Try it" playground on every endpoint.

```ts
// Everything is just MDX + an OpenAPI file.
export const hello = "world";
```

## Next steps

- Edit `docs/index.mdx` (this page) and `docs/quickstart.mdx`.
- Point `api.baseUrl` in `docs.json` at your API and drop your spec in
  `api/openapi.json`.
- Run `markline build` for a server bundle, or `markline export` for static HTML.
